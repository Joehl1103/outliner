# Outliner MVP

Workflowy-style outliner MVP wireframe built with Next.js.

## Current Scope

- Static wireframe rows (read-only)
- Browser persistence via `localStorage`
- Seeded default outline on first load

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Verify

```bash
npm test
npm run build
```
